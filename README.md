Fall In Love With CSS
========================

CSS is an ancillary technology for many software developers,
similar to other declarative languages used for specific tasks.
Historically there's been a deep mistrust of the language,
embedded in jokes and so many projects that try to remove the need to write any CSS.

Let me show you how to think about CSS.
We'll cover the concepts,
talk about the cascade,
and show off some of the best features of modern CSS.

Spoilers:

* Variables (custom properties)
* nested styles (CSS nesting)
* Two! Great layout tools (CSS grid and flexbox)
